A user reported an unexpected email attachment named invoice.pdf. Your task is to confirm its true file type and generate a hash for threat intelligence lookup.

  1. Launch the VM: Click the **VM Running** button on the left to start your remote desktop session (if not already running).
  2. **Transfer File:** Imagine transferring the file invoice.pdf to your VM's desktop for analysis.
  3. **File Type Identification:** Open a terminal in the VM and use the Linux file utility to determine the file's true signature: 
    file ~/Desktop/invoice.pdf
  4. **Hashing:** Use the sha256sum command to generate a cryptographic hash of the file for intelligence database checks: 
    sha256sum ~/Desktop/invoice.pdf
  5. **String Analysis:** Use the strings command to look for embedded text or malicious code snippets.

Task Summary

Complete the analysis inside the VM. Record the output of the file command and the SHA256 hash.
