Metadata Investigation: Revealing Hidden Informations

Metadata investigation adalah proses pemeriksaan informasi tersembunyi yang melekat pada sebuah file, seperti tanggal pembuatan, perangkat yang digunakan, lokasi geografis, versi aplikasi, hingga histori perubahan. Metadata sering kali menjadi kunci utama dalam proses digital forensic karena dapat mengungkap konteks yang tidak terlihat dari isi file itu sendiri. Misalnya, sebuah gambar mungkin terlihat biasa, tetapi metadata-nya dapat menunjukkan GPS lokasi foto diambil, model kamera, timestamp, bahkan software yang memodifikasinya.

Dalam investigasi digital, metadata membantu analis memverifikasi keaslian file, membangun timeline kejadian, mengidentifikasi pelaku atau perangkat yang digunakan, dan mendeteksi upaya manipulasi. Metadata juga sangat penting ketika menghadapi kasus pencurian data, pemalsuan dokumen, investigasi kejadian kriminal, hingga serangan siber. Dengan memanfaatkan tools forensik seperti exiftool, mat2, atau exiv2, metadata dapat dianalisis secara lebih cepat, akurat, dan menyeluruh.

Ruang Lingkup Metadata Investigation

Kategori Metadata Contoh Informasi Manfaat Investigasi
File System Metadata Timestamp (Created, Modified, Accessed), file owner, permission Membangun timeline kejadian dan memvalidasi apakah file dimodifikasi
Document Metadata Author, revision history, application version Menemukan siapa pembuat dokumen dan mendeteksi manipulasi atau editing
Image Metadata (EXIF) GPS location, camera model, lens info, date-taken Mengidentifikasi lokasi kejadian dan verifikasi keaslian foto
Application Metadata Software signatures, embedded versioning Melacak sumber pembuatan file, menemukan aplikasi yang digunakan
Network Metadata IP address, host information Melacak sumber akses atau aktivitas jaringan
Embedded Metadata Hidden strings, checksum, internal markers Menemukan indikasi malware, atau hidden content

Tools for Metadata Investigation

Tools Fungsi Jenis File
ExifTool Membaca seluruh metadata, termasuk EXIF, IPTC, XMP, dan lainnya. Gambar, dokumen, video, audio
Exiv2 Analisis dan modifikasi metadata gambar Gambar (JPG/RAW)
Mat2 (Metadata Anonymization Toolkit) Analisis dan menghapus metadata sensitif Office, PDF, gambar
PDFinfo / PDFid Melihat metadata dan properti PDF PDF
Strings Mengambil string tersembunyi—indikator malware File biner
